# Інструкція з міграції бази даних

Виконайте наступні команди в терміналі в папці server:

## 1. Створити міграцію

```bash
npx prisma migrate dev --name add_routine_categories
```

## 2. Запустити seed для додавання категорій

```bash
npx prisma db seed
```

## Альтернатива: якщо потрібно скинути БД

```bash
npx prisma migrate reset
```

Це:

- Очистить базу даних
- Застосує всі міграції заново
- Запустить seed автоматично
